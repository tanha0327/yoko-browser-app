# Yoko Browser App

## 概要
Yoko Browser Appは、横画面専用のスマートフォンブラウザアプリです。このアプリは、ユーザーが快適にウェブコンテンツを閲覧できるように設計されています。

## 特徴
- 横画面専用のインターフェース
- URLの入力とページ表示を管理するブラウザコンポーネント
- デバイスの向きを検出するカスタムフック
- スタイリッシュでレスポンシブなデザイン

## インストール
このプロジェクトをローカルにクローンして、依存関係をインストールするには、以下のコマンドを実行してください。

```bash
git clone <repository-url>
cd yoko-browser-app
npm install
```

## 使用方法
アプリケーションを起動するには、以下のコマンドを実行します。

```bash
npm start
```

ブラウザが自動的に開き、アプリケーションが表示されます。

## 開発
このプロジェクトはTypeScriptで構築されており、Reactを使用しています。コードの変更を行った後は、アプリケーションを再起動することなく、変更が即座に反映されます。
